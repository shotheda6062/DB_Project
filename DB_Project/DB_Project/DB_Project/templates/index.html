國際轉運使用教學


第一步：註冊拿地址


綁定email註冊會員，我們將提供海外倉庫地址（英、美、日、韓），還可合併運單一起打包送台灣。

第二步：全球剁手指


開始盡情在海外網購，送貨地址填寫我們海外倉庫地址（收貨人記得加上U開頭的用戶編號），讓我們為您簽收戰利品吧！

第三步：乖乖來申報


賣家寄出貨品，以物流編號（Tracking number）申報貨件資訊，物流配送透明，全程網站查詢看的到。

第四步：填寫轉運單


貨品抵達香港倉庫，確認重量，填寫轉運單合併包裹。

第五步：翹腳等收貨


付款完成，倉庫出貨到台灣，運費貨到付款快捷又可靠。


運費價格


所有倉庫統一價格：首重 NT$ 150, 續重每千克 NT$ 150


按包裹實際重量收費 *價錢已包含國際空運及宅配到府的費用


若產生台灣應繳關稅者，我們將全額為您負擔

What we offer


每天出庫

我們所有海外倉庫均會每天出庫，貨件2-6天內運抵香港倉庫，再讓你合併運單轉送台灣。

實重收費

多國貨件統一收費標準，以貨件送達香港倉庫時的總重量計算價格，清楚易懂。

物流狀況即時更新

貨件物流狀況如到達海外倉庫、運抵香港、安排派送等，均可在網站系統上查詢。

簡單取件方式

貨件到達台灣後宅配到府，運費貨到付款，快捷可靠。
